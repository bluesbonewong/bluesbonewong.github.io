CSS3相关面试题

发表于 2018-05-09 | 分类于 面试题

¶1. 说说盒模型

  • content-box: width === 内容区宽度
  • border-box: width === 内容区宽度 + padding 宽度 + border 宽度(不管IE，都用border-box)

¶2. css reset 和 normalize.css 有什么区别？

reset 重置，之前的样式我不要，a{color: red;}，抛弃默认样式 normalize 让所有浏览器的标签都跟标准规定的默认样式一致，各浏览器上的标签默认样式基本统一

¶3. 如何居中

¶水平居中：

  • 内联：爸爸身上写 text-align:center;
  • 块级：margin-left: auto; margin-right: auto;

¶垂直居中：

  • flex justify-content:center align-items:center
  • display:position(absolute/fixed) + transform/margin:auto/margin-top:-xxxpx
  • 行内元素居中，height = line-height
  • 元素外设置table标签，margin设置为auto

¶4. 选择器优先级如何确定？

选择器越具体，优先级越高。 #xxx 大于 .yyy 同样优先级，写在后面的覆盖前面的。 color: red !important; 优先级最高。

¶5. BFC 是什么？

overflow:hidden 清除浮动。（一般是用 .clearfix 清除浮动，坚决不用 overflow:hidden 清除浮动） overflow:hidden 取消父子 margin 合并。http://jsbin.com/conulod/1/edit?html,css,js,output （一般用 padding-top: 1px;）

¶6. 如何清除浮动？

  • overflow: hidden （方方反对）
  • .clearfix 清除浮动写在爸爸身上

¶7. CSS引入的方式有哪些? link和@import的区别是?

  • 写在HTML里的Style属性里
  • 写在Style标签里
  • 使用引入Style.css
  • 在Style标签内使用@import url(style.css)导入

¶区别

  1. link不仅可以引入样式文件，还可以做其他事情比如定义rss等；@import只能用来导入css
  2. link是页面载入同时加载；@import是页面完全载入后再加载
  3. link引入的样式可以使用js去操作dom改变样式；@import不支持
