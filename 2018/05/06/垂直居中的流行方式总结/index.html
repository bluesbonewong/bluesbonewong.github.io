¶总结

所以，我们一共有六种方式来实现我们的需求，当然还有更多的方法，大家可以自行探索

  1. 直接设置.father的padding，限制：父级元素height不能写死，子元素width不能写死
  2. position:absolute(fixed) + margin-top/margin-left，限制：要求子元素宽高固定，否则margin-top/margin-left无法进行计算
  3. position:absolute(fixed) + transform，限制：transform不兼容低版本IE
  4. position:absolute(fixed) + margin:auto，限制：要求子元素宽高固定，否则会自动撑满父容器
  5. flexbox，限制：flexbox不兼容低版本IE
  6. vh + transform，限制：使用场景有限，仅限于某一个元素需要在当前窗口居中，实现方式与position方法类似
